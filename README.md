# openci

Simple tool to open the ci build url of a repository.


## Installation
```bash
go get -u github.com/thrawny/openci
```

## Usage
```bash
cd /path/to/project/with/ci
openci
```

## Supported CI providers
- CircleCI
- TravisCI
- Wercker


## Contributing

Feel free to submit a PR and add support for a new CI provider.
